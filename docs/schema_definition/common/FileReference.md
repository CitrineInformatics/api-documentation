## Description

A reference to a file.

## Fields

Field name | Value type | Description
-----------|------------|------------
`relativePath` | String | String with the relative path of file.
`mimeType` | String | String with the mime type of the file.
`sha256` | String | String with the SHA-256 hash of the file.
`md5` | String | String with the MD5 hash of the file.
`tags` | Array of strings | Tags that apply to the ID.
